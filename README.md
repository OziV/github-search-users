# Github Search Users

> React search Github users app.

## Site URL

[Github Search Users](https://oziv-github-search-users.netlify.app/)

## Technologies

- Frontend
  - ReactJS

## Deployment

[Netlify](https://www.netlify.com/)

## Usage

### Install Dependencies

```
npm install
```

### Run

```
npm run start
```
